Глобальные настройки (GlobalSettings) — таблица для глобальных настроек приложения.

Пользователи (User) — таблица для пользователей.
Настройки пользователя (UserSettings) — таблица для персонализированных настроек пользователей.

Задачи (Task) — основная таблица для хранения информации о задачах.
Статусы задач (TaskStatus) — таблица для статусов задач.
Приоритеты задач (TaskPriority) — таблица для уровней приоритета задач.
Категории задач (Category) — таблица для категорий задач.
Типы задач (TaskType) — таблица для типов задач.
История изменений задачи (TaskHistory) — таблица для истории изменений задач.
Комментарии к задачам (TaskComment) — таблица для комментариев к задачам.
Права доступа задачи (TaskPermission) — таблица для прав доступа к задачам (например, кто может редактировать, просматривать или удалять задачи).
TaskSettings (Настройки задачи)

Статистика по задачам (TaskStatistics) — таблица для статистики по задачам.
Статистика пользователя (UserStatistics) — таблица для статистики по пользователю.

Команды (Team) — таблица для команд (групп пользователей).
Члены команды (TeamMember) — таблица для связи пользователей с командами.
Права доступа команды (TeamPermission) — таблица для прав доступа, связанных с командой (например, разрешение на доступ к задачам команды).
